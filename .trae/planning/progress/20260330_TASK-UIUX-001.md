# UI/UX 优化任务计划

## 任务概述
对 AI 调度中心控制台首页进行视觉优化，提升整体美观度和用户体验。

## 需求分析

### 1. Logo 颜色优化
**问题：** 左上角 Logo 的绿色太丑，需要改浅一点
**当前：** `linear-gradient(135deg, #22C55E 0%, #16A34A 100%)` - 深绿色
**目标：** 使用更浅的绿色，提升视觉舒适度

### 2. Div 间隔紧凑化
**问题：** 页面内容区域的间隔过大，需要紧凑布局
**当前状态：**
- `.main-content` padding: 28px
- `.welcome-section` margin-bottom: 32px
- `.overview-section` gap: 20px, margin-bottom: 28px
- `.action-section` gap: 24px
- `.stat-card` padding: 24px
- `.action-panel/.guide-panel` padding: 24px

**目标：** 调整间距使页面刚好在可视区域内完整展示，无需滚动

### 3. 背景玻璃感增强
**问题：** 背景的玻璃感不明显，需要灰黑渐变增强视觉效果
**当前：** 
```css
background: 
  radial-gradient(ellipse 80% 50% at 20% 40%, rgba(34, 197, 94, 0.08) 0%, transparent 50%),
  radial-gradient(ellipse 60% 40% at 80% 60%, rgba(34, 197, 94, 0.05) 0%, transparent 50%),
  #0F172A;
```
**目标：** 添加灰黑渐变层次，增强玻璃拟态效果

### 4. 非线性内容平滑切换
**问题：** 内容切换需要更流畅的过渡动画
**目标：** 实现页面内容切换时的平滑过渡效果

## 技术方案

### 修改文件
- `frontend/src/views/Overview.vue` - 主控制台页面

### 具体修改项

#### 1. Logo 颜色调整
```css
/* 当前 */
.logo-icon {
  background: linear-gradient(135deg, #22C55E 0%, #16A34A 100%);
}

/* 改为更浅的绿色 */
.logo-icon {
  background: linear-gradient(135deg, #4ADE80 0%, #22C55E 100%);
  box-shadow: 0 4px 14px rgba(74, 222, 128, 0.4);
}
```

#### 2. 紧凑布局调整
```css
/* 主内容区 */
.main-content {
  padding: 16px 20px;  /* 从 28px 减少 */
}

/* 欢迎区域 */
.welcome-section {
  margin-bottom: 16px;  /* 从 32px 减少 */
  padding: 12px 0;      /* 从 24px 减少 */
}

/* 概览卡片区域 */
.overview-section {
  gap: 12px;            /* 从 20px 减少 */
  margin-bottom: 16px;  /* 从 28px 减少 */
}

/* 统计卡片 */
.stat-card {
  padding: 16px;        /* 从 24px 减少 */
  border-radius: 16px;  /* 从 20px 微调 */
}

/* 操作区域 */
.action-section {
  gap: 16px;            /* 从 24px 减少 */
}

/* 面板 */
.action-panel,
.guide-panel {
  padding: 16px;        /* 从 24px 减少 */
}
```

#### 3. 背景玻璃感增强
```css
.main-content {
  background: 
    /* 灰黑渐变层 - 增强玻璃感 */
    linear-gradient(135deg, rgba(30, 41, 59, 0.4) 0%, rgba(15, 23, 42, 0.8) 50%, rgba(30, 41, 59, 0.6) 100%),
    /* 原有的绿色光晕 */
    radial-gradient(ellipse 80% 50% at 20% 40%, rgba(34, 197, 94, 0.06) 0%, transparent 50%),
    radial-gradient(ellipse 60% 40% at 80% 60%, rgba(34, 197, 94, 0.04) 0%, transparent 50%),
    /* 基础深色背景 */
    linear-gradient(180deg, #1E293B 0%, #0F172A 50%, #1E293B 100%);
}
```

#### 4. 平滑过渡动画
```css
/* 页面切换过渡 */
.page-layout {
  transition: opacity 0.3s ease, transform 0.3s ease;
}

/* 卡片悬停优化 - 更平滑 */
.stat-card {
  transition: all 0.4s cubic-bezier(0.4, 0, 0.2, 1);
}

/* 内容区域淡入动画 */
.main-content > * {
  animation: fadeInUp 0.5s ease-out forwards;
}

@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* 交错动画延迟 */
.welcome-section { animation-delay: 0s; }
.overview-section { animation-delay: 0.1s; }
.action-section { animation-delay: 0.2s; }
```

## 验收标准

1. **Logo 颜色：** 使用浅绿色渐变，视觉效果更舒适
2. **布局紧凑：** 整个页面内容在 1080p 屏幕上无需滚动即可完整展示
3. **玻璃感：** 背景有明显的灰黑渐变层次，玻璃拟态效果明显
4. **平滑切换：** 页面元素有流畅的淡入和悬停过渡效果

## 执行步骤

1. 修改 Logo 颜色为浅绿色渐变
2. 调整所有间距参数实现紧凑布局
3. 增强背景渐变层次
4. 添加平滑过渡动画
5. 验证所有修改在深色和浅色模式下都正常
