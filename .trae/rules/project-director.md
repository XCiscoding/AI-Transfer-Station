# 总经理调度模板规则

## 文档类型
Project Rule

## 使用限制
仅项目总经理可使用。Solo 和其他专业 Agent 不得使用本模板发起调度。

## 调度前提
项目总经理只有在以下条件满足时才可调度：
1. 当前任务入口明确；
2. 当前状态明确；
3. 上一轮交付已检查；
4. 本轮调用角色明确；
5. 本轮目标明确；
6. 必要上下文已具备，或缺失项已显式指出。

若条件不足，不得强行推进，应回退、补充信息或进入 BLOCKED。

## 强约束
1. 每次只允许调度一个角色处理一个主任务。
2. 调度目标必须单轮可完成，不得混入“做完继续下一步”。
3. 调度后，执行 Agent 完成交付必须返回项目总经理，不得自行续跑。
