### API 测试端点

```bash
# 登录获取Token
TOKEN=$(curl -s -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}' | jq -r '.token')

# 获取渠道列表
curl http://localhost:8080/api/v1/channels -H "Authorization: Bearer $TOKEN"

# 获取真实Key列表
curl http://localhost:8080/api/v1/real-keys -H "Authorization: Bearer $TOKEN"

# 获取虚拟Key列表
curl http://localhost:8080/api/v1/virtual-keys -H "Authorization: Bearer $TOKEN"
```
